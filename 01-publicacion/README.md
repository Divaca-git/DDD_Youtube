# Contexto 1 — Publicación y distribución

Ciclo de vida técnico del video: subida, procesamiento, reproducción y transmisión en vivo.
El "video" aquí es un **asset técnico** (sin metadata editorial).

- `openapi.yaml` — especificación de la API (incluye la descripción de responsabilidad)
- `diagrama-clases.drawio` — modelo conceptual
- `diagrama-secuencia.drawio` — flujo "de la subida al catálogo"

Eventos que emite: ContenidoSubido, ProcesamientoFinalizado, AssetListoParaPublicacion,
SesionReproduccionCompletada, TransmisionIniciada, TransmisionFinalizada.
